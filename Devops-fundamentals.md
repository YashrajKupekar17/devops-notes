
# Lecture 2 – DevOps Fundamentals

## 1. What is DevOps?
DevOps is a **culture + practices + tools** that combines Development and Operations.

Goal:
- Faster delivery
- Higher reliability
- Continuous improvement

---

## 2. Why DevOps?

Key benefits:
- Faster releases
- Better collaboration
- Automation
- Scalability
- Continuous feedback

---

## 3. DevOps Lifecycle

Plan → Code → Build → Test → Release → Deploy → Operate → Monitor

This cycle is **continuous**, not linear.

---

## 4. Core DevOps Principles
- Collaboration
- Automation
- CI/CD
- Infrastructure as Code
- Monitoring & feedback
- Measurement

---

## 5. DevOps Practices

- Continuous Integration
- Continuous Delivery / Deployment
- Automated testing
- Infrastructure as Code
- Monitoring & logging
- Configuration management

---

## 6. Common DevOps Tools

| Stage | Tools |
|----|-----|
| Version Control | Git |
| CI/CD | Jenkins, GitHub Actions |
| Containers | Docker |
